# ATEM_Controller

MIDIパッドでATEMを操作できるようにするソフトです．
 
# 使い方

ATEM-Controller.pyを実行するだけです．
'''./Profiles/'''にあるCSVファイルを書き換えて，ご自由にご利用ください．

なお，当ソフトを使用した際のいかなる問題に対しても保証できかねますので，自己責任でお願いします．

初期プロファイルは
* ip : 192.168.10.240
* MIDI pad : Novation Launchpad X 
に対応しています．  
 
# 動作環境
 
* Python 3.7~
* PyATEMMax 1.05b~
* mido 1.2.10~
